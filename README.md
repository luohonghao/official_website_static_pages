# pinzaSite

#### 介绍

品宅官网页面

#### 软件架构

纯静态前端页面+后端RestfulAPI

#### 安装教程

可选步骤

1. 安装`Nodejs`
2. 安装`browser-sync`,自动刷新页面工具
``` bash
npm install -g browser-sync
```

#### 使用说明

1. 启动自动刷新页面服务
```` bash
browser-sync start --server --files "**/\*.css, **/\*.html"
````
